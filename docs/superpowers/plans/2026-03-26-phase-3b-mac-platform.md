```markdown
# RemoteFlow — App Store Description

## Subtitle (30 chars max)
SSH Terminal, Scroll Freedom

## Promotional Text (170 chars max)
The SSH terminal that respects your scroll position. Read AI output, build logs, and server streams without fighting auto-scroll.

## Description

RemoteFlow is a modern SSH terminal client built for developers who work with remote servers.

**The Problem:** Every mobile SSH client auto-scrolls to the bottom when new output arrives. When you're monitoring a build, tailing logs, or watching AI-assisted development sessions, you can't scroll up to read previous output without the terminal snapping back down.

**The Solution:** RemoteFlow's scroll-aware streaming detects when you've scrolled up and holds your position. New output continues arriving in the background. A subtle badge tells you when there's new content below. Tap it or scroll to the bottom to resume auto-scroll.

**Features:**
- Scroll-aware terminal streaming — read at your own pace
- SSH password and key authentication
- Up to 4 concurrent terminal sessions
- Session dashboard with live terminal previews
- Dark and light themes with 4 terminal color schemes
- 10,000-line scrollback buffer
- Copy and paste with multi-line support
- Network disconnect detection and recovery
- iPad multitasking (Split View, Slide Over)
- VoiceOver and Dynamic Type accessibility
- No tracking, no analytics, no ads

**Built for:**
- Developers monitoring CI/CD pipelines
- DevOps engineers tailing production logs
- Anyone who SSH's from their phone or tablet

RemoteFlow is open source: github.com/thegeshwar/remoteflow

## Category
Utilities > Developer Tools

## Age Rating
4+
```
